It is a final project from Colt Steele's "The Web Developer Bootcamp 2022" where you will find and interactive map with campgrounds in USA. App is built using JavaScript, Express, MongoDB and other middleware. It also has user authentication, authorization, CRUD Operations and follows RESTful convention. It is a responsive online platform with such functionalities:
- User can register / login / logout;
- User can add, delete, edit, comment, add photo and rate Campgrounds;
- User can use an interactive map to find a campground.


# Built with

Front-end:

- Bootstrap

Back-end:
- express
- mongoDB
- mongoose
- async
- passport
- passport-local
- express-session
- method-override
- nodemailer
- moment
- geocoder
- connect-flash


Start app by writing in Terminal ```nodemon app.js```
